## Install

```bash
composer require awema-pl/indigo-layout
```

## Example
```blade
@extends('indigo-layout::auth')
 
@section('title')
    <h1>Awesome App!</h1>
@endsection
 
@section('content')
    ...
@endsection
 
@section('footer')
    ...
@endsection
```
